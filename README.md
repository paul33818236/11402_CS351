# README

這個存放庫用於 CS351 課程相關資源與作業。

## 課程介紹

CS351 是一門教授計算機科學基礎與應用的課程，內容涵蓋程式設計、資料結構、演算法與軟體工程等主題。

## 目錄結構

- `assignments/`：課堂作業與練習題。
- `notes/`：課堂筆記與參考資料。
- `projects/`：課程專題或範例程式碼。

## 使用說明

1. 克隆此存放庫到本地：
   ```bash
   git clone https://github.com/yourusername/11402_CS351.git
   ```
2. 瀏覽各資料夾中的檔案，參考與學習。
3. 按課堂要求提交作業或更新內容。

## 聯絡方式

有問題請聯絡授課教師或助教。
